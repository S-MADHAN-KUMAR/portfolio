# Dev Portfolio | Cursor IDE Clone

A premium, interactive developer portfolio built with Next.js, mimicking the experience of the Cursor IDE.

## Features

- **Interactive IDE Interface**: Explorer, Editor, and AI Chat panels.
- **Resizable Layout**: Horizontally adjustable sidebars and chat panels.
- **Material Icon Theme**: Custom styled icons and folder structures.
- **AI Chat Simulator**: Ask questions about my experience and skills.
- **Preview/Code Views**: Toggle between high-end UI previews and raw code.

## Tech Stack

- **Framework**: [Next.js 14+](https://nextjs.org)
- **Styling**: Vanilla CSS with CSS Grid & Variables
- **Icons**: [Lucide React](https://lucide.dev)
- **Animations**: CSS Transitions & Keyframes

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000)

## Project Structure

- `src/app/page.tsx`: Main IDE component and logic.
- `src/app/globals.css`: Core design system and IDE styling.
- `portfolioData`: Centralized mock data for the explorer.
